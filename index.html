Frequently Asked Questions

What is the differences between var, let and const?

1. The difference between var, let, and const
var: Old way of declaring variables, function-scoped, can be redeclared.
let: Block-scoped, can't be redeclared but can be updated.
const: Block-scoped, can't be redeclared or updated (but objects & arrays can be modified).

What is the differences between map(), forEach() and filter()?

2. The difference between map(), forEach(), and filter()
map(): Returns a new array after modifying each element.
forEach(): Loops through each element but doesn’t return a new array.
filter(): Returns a new array with elements that meet a condition.

What are arrow functions, and how are they different from regular functions?

3. Arrow functions vs. Regular functions
Arrow functions (=>): Shorter syntax, don’t have their own this, and can’t be used as constructors.
Regular functions: Have their own this and can be used with new.

How do JavaScript Promises work?

4. How JavaScript Promises work
A Promise is like a guarantee that a task (like fetching data) will complete in the future. It has three states: Pending (waiting) Resolved (successful) Rejected (failed).

How does closure work in JavaScript?

5. How closures work in JavaScript
A closure happens when a function remembers variables from its outer scope even after the outer function has finished running.
